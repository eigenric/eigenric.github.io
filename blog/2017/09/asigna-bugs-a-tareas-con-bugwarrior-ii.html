Asigna bugs a tareas con Bugwarrior (II)

2 sep 2017

Bugwarrior es una herramienta excelente para gestionar issues de GitHub y otros servicios como Trello. Cuenta con una documentación completa y está escrito en Python, lo que facilita la contribución al proyecto.

Instalación

Para instalar Bugwarrior y las dependencias necesarias, usa el siguiente comando:

$ pip install "bugwarrior[keyring]"

Esto incluye las dependencias necesarias para usar bugwarrior-vault, la herramienta que gestiona las claves.

Crea un archivo .config/bugwarrior/bugwarriorrc con el siguiente contenido, sustituyendo eigenric por tu nombre de usuario de GitHub:

[general]
targets = eigenric_github_account

[pwaqo_github_account]
service = github
github.login = eigenric
github.username = eigenric
github.password = @oracle:use_keyring
github.description_template = {{githubnumber}}:{{githubtitle}}

El campo github.password = @oracle:use_keyring indica que la clave se almacenará utilizando bugwarrior_vault

Almacenamiento de Claves

Registra la contraseña en el anillo de claves con el siguiente comando:

$ bugwarrior-vault set github://eigenric@github.com/eigenric eigenric

Introduce la contraseña cuando se te solicite.

Filtros Personalizados

Es posible configurar Bugwarrior para obtener únicamente las issues que cumplan ciertos requisitos, como:

  • Asignación propia
  • Etiquetas específicas
  • Proyectos específicos
  • Ejecución de la Sincronización

Para ejecutar la sincronización y obtener las issues, utiliza el siguiente comando:

$ bugwarrior-pull

Esta configuración te permitirá gestionar tus issues de GitHub de manera eficiente y personalizada, aprovechando la flexibilidad y potencia de Bugwarrior.
