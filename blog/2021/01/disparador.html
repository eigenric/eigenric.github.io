Disparador

21 ene 2021

Nos disponemos a esbozar el disparador de la restricción semántica 4.4.2. El producto a valorar debe aparecer en el historial de compra del cliente.

La tabla a la que iría vinculado seria la tabla de valoración.

En primer lugar, debemos especificar que es el historial de compra del cliente. Disponemos de CarritoPagado y CarritoFinalizado. Entiendo que el historia ya es lo que ya está pagado.

Antes de que un cliente con cierto DNI valore un producto debemos comprobar si el carritocliente cuyo IDcompra se encuentre en carritopagado.

¿Cómo obtener el Historial de compra?

Disponemos de las tablas CarritoCliente. CarritoPagado. DetallesCarrito. Necesitamos los DetallesCarrito de todos los CarritosPagados

Se nos ocurre lo siguiente:

SELECT x.nombre, x.artista, y.unidadescomprar FROM PRODUCTO x, DetallesCarrito y WHERE x.nref = y.nref AND y.idcompra IN (SELECT idcompra FROM CarritoPagado)

Estor resulta en los nombre, artista de los productos pagados.

SELECT x.nombre, x.artista, z.dni FROM Producto x, DetallesCarrito y, CarritoCliente z WHERE x.nref=y.nref AND z.dni= AND z.idcompra=y.idcompra AND y.idcompra IN (SELECT idcompra FROM Carritopagado)

Actualmente hemos conseguido obtener el historial de compra mediante INNER JOINS DE 3 Tabals:

select d.nombre, d.artista from CarritoCliente a inner join CarritoPagado b on a.idcompra = b.idcompra inner join DetallesCarrito c on b.idcompra = c.idcompra inner join Producto d on d.nref = c.nref WHERE a.DNI = '97867564F'

Nos falta el disparador, que lanza error si no esta en el historial.

Esbozo del disparador:

INSERT INTO Valoracion (DNI, NRef, Valoracion) values ('14443545G','0000002-V',4); CREATE OR REPLACE TRIGGER no_puede_valorar BEFORE INSERT ON Valoracion FOR EACH ROW
BEGIN IF (:new.NRef NOT IN (SELECT nref INTO historial_compra FROM CarritoCliente a inner join CarritoPagado b on a.idcompra = b.idcompra inner join DetallesCarrito c on b.idcompra = c.idcompra WHERE a.DNI = :new.DNI)) THEN RAISE_APPLICATION_ERROR(-20603,'No se puede valorar un producto que no ha sido comprado'); END IF; END;

Me responden de StackOverflow:

CREATE OR REPLACE TRIGGER NOPUEDEVALORAR BEFORE INSERT ON Valoracion FOR EACH ROW
DECLARE l_exists NUMBER(1) := 0; BEGIN SELECT MAX(1) INTO l_exists FROM dual WHERE EXISTS (SELECT nref FROM CarritoCliente a INNER JOIN CarritoPagado b ON a.idcompra = b.idcompra INNER JOIN DetallesCarrito c
ON b.idcompra = c.idcompra WHERE a.dni = :NEW.dni AND c.nref = :NEW.nref ); IF l_exists = 0 THEN RAISE_APPLICATION_ERROR(-20603,'Cant rate a not bought product'); END IF; END;
