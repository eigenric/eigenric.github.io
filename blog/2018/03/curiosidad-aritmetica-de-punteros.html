Curiosidad aritmética de punteros

6 mar 2018

Restar punteros devuelve la distancia entre las posiciones de los punteros en memoria. Por ejemplo, si tenemos dos punteros p y q que apuntan a posiciones consecutivas de un arreglo, entonces p - q es igual a 1.

#include <stdio.h>

int main() {
    int a[5] = {1, 2, 3, 4, 5};
    int *p = &a[0];
    int *q = &a[1];

    printf("%d\n", p - q); // -1
    printf("%d\n", q - p); // 1

    return 0;
}
