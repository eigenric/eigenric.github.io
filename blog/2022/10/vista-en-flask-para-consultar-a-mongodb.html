Vista en Flask para consultar a MongoDB

28 oct 2022

En el contexto de la asignatura DAI (Desarrollo de Aplicaciones para Internet), se nos pide realizar una pequeña aplicación web que consulte una base de datos de recetas de cócteles. La base de datos es MongoDB y la aplicación web se construye con Flask en la primera versión y Flask Restful en la segunda.

Unos de los ejercicios consiste en elaborar una vista que devuelva las recetas que contengan cierto ingrediente. Los ingredientes son una lista interior a las recetas. Vodka es un elemento de los ingredientes.

El truco es hacer ingredients.name y luego comprobar que son iguales con el operador $eq

@app.route("/recetas_con/<bebida>")
def mongo_recetas_con(bebida):
    """Devuelve las recetas con cierta bebida en sus ingredientes"""

    recetas_con_bebida = db.recipes.find({
        "ingredients.name": {"$eq": str(bebida) }
    } )    

    lista_recetas = []
    for receta in recetas_con_bebida:
        app.logger.debug(receta)
        lista_recetas.append(receta)

    response = {
        "len": len(lista_recetas),
        "data": lista_recetas
    }

    resJson = dumps(response)

    return Response(resJson, mimetype="application/json")
